## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

[![CircleCI](https://circleci.com/gh/MittyP/circleci-test.svg?style=shield)](https://circleci.com/gh/MittyP/circleci-test)
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
